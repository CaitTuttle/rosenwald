Concept #2::

School rankings have been used to punish (to put it simply) poor, Black schools. School rankings need to be more comprehensive, and it is necessary to address structural inequities Define Structural Inequities to improve school outcomes.

Takeaways

*Please note that the Equity Rankings are not intended to be a complete analysis of school success. It is rather created to start conversations about how county rankings that use gross outcomes (i.e. test scores, graduation rates) are not an accurate measure of success, but rather a measure of affluence. A more comprehensive measurement is necessary.
